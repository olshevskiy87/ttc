tiny typos checker
==================

dependencies
------------

perl modules:
- Term::ANSIColor
- File::Find
- Getopt::Long

usage
-----

just run

```sh
./ttc.pl --path=../path/to/files
```

if you have your own file with wrong words run

```sh
./ttc.pl --ww_path=my_wrong_words.txt --path=.
```
